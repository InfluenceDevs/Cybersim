# Learning Paths

CyberSim now includes a guided progression model with role-aware training.

## Team modes

- Blue Team (defender): prioritize detection, triage, and mitigation outcomes.
- Red Team (attacker): prioritize exploitation path discovery and attack chaining.

## Paths

## Path 1: Recon Foundations (Beginner)

- Fake terminal command drills
- File system exploration and secret hunting
- Basic network scan interpretation

## Path 2: Web Exploitation Basics (Intermediate)

- Vulnerable login workflow (SQLi, weak passwords, no rate-limit behavior)
- Reflected XSS lab
- CSRF simulation
- File upload bypass simulation

## Path 3: Defensive Response (Intermediate)

- Payload playground with blocked vs executed simulation
- Instructor rubric tuning
- Detection-focused debrief interpretation

## Path 4: Operator Assessment (Advanced)

- Instructor sessions
- Checkpoint-based scoring
- JSON/CSV export reporting
- Readiness trend interpretation

## Gamification

- XP and levels for challenge completion
- Badge unlocking for practical milestones
- Weak area tracking for focused remediation
