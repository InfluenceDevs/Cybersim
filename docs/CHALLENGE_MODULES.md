# Challenge Modules

This document maps each challenge module to educational outcomes.

## Fake Terminal Scenarios

- Find password in files
- Exploit weak permissions
- Decode base64 strings

Outcome: command-line triage and artifact discovery.

## Vulnerable Login Lab

- SQL Injection bypass simulation
- Weak password acceptance simulation
- Profile switching (vulnerable vs hardened)

Outcome: authentication control design and attack validation.

## Vulnerable Mini Websites

- Reflected XSS behavior
- CSRF action without token
- File upload bypass with disguised executable names

Outcome: understand common web flaws and practical defenses.

## File System Explorer

- Search and open fake system files
- Locate hidden secrets in `.env` and config assets

Outcome: evidence collection and misconfiguration discovery.

## Phishing Detection Game

- Identify suspicious domains and urgency patterns
- Classify safe vs malicious emails

Outcome: human-layer detection and incident triage.

## Password Cracking Simulator

- Dictionary vs brute-force strategy simulation
- Progress and result interpretation

Outcome: password policy awareness and cracking risk literacy.

## Network Scan Page

- Parse fake port scan output
- Choose attack path based on exposed services

Outcome: recon analysis and attack surface prioritization.

## Payload Playground

- Simulate blocked/executed outcomes
- Compare red-team and blue-team perspectives

Outcome: payload reasoning and defensive filtering logic.

## Progress Tracker

- XP, levels, badges, weak-area indicators
- Completion status and focus recommendation

Outcome: sustained training and measurable growth.
